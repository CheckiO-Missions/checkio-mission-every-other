Every other element of a give array should be changed to 0, starting with the first one.

Input: {% if interpreter.slug == "js-node" %}Array{% else %}Iterable{% endif %}

Output: {% if interpreter.slug == "js-node" %}Array{% else %}Iterable{% endif %}

Example:

{% if interpreter.slug == "js-node" %} 
everyOther([1, 2, 3, 4, 5]) == [0, 2, 0, 4, 0]
everyOther([0, 0, 0]) == [0, 0, 0]
{% else %}
